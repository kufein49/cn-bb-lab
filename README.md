# cn-bb-lab

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库用于存放和展示各类独立 HTML 页面。这些页面可能包含实验性项目、工具页面、信息展示页或其他前端小作品。仓库本身不针对任何特定域名或网站，只作为页面文件的集合与归档空间。

每个页面均为自包含的 HTML 文件，可独立运行，无需额外依赖。

## 目录说明

```
cn-bb-lab/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面可能用到的通用资源（如图片、样式、脚本）
├── index.html      # 仓库首页（可选，用于导航或列表展示）
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或单个 HTML 文件代表一个独立页面。命名尽量简洁，反映页面用途。
- **assets/**：如果多个页面共享资源，可统一放置于此。页面也可以自带资源文件。
- **index.html**：可选的仓库首页，用于快速浏览或跳转到各个归档页面。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容、样式、脚本均内联或随文件一起提交，确保离线可查看。
- 页面不收集用户信息，不包含第三方跟踪代码。
- 归档的页面可能包含实验性功能或临时内容，不保证长期可用性或兼容性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 欢迎提出 Issue 或 Pull Request，但请注意：仓库不接受与页面内容无关的请求。
- 页面内容应符合开源社区规范，不得包含违法、侵权或恶意内容。
- 对于长期未维护或已失效的页面，可能会被清理或归档到 `archive/` 目录。
